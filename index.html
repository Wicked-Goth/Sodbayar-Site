Overview

What this project includes

The assignment asked for multiple HTML pages, lists, a table, images, links, rich form controls, CSS in several forms, and JavaScript plus jQuery. This site now includes each of those pieces.

Highlights
  1. A responsive navigation menu with desktop and mobile states.
  2. A gallery page with images, lists, and a styled comparison table.
  3. A contact page with full jQuery form checking and feedback.
Project note

I kept Tailwind as the main styling system, then added a small shared CSS file and page-level style tags so the project matches the assignment without breaking the original design direction.
