# Sodbayar-Site

Multi-page personal portfolio and final project website for Web Technologies.

## Files

- `index.html`: homepage and project overview
- `story.html`: personal story, ordered list, and bulleted list
- `gallery.html`: images, links, animated notes, and a formatted table
- `contact.html`: full form with jQuery validation
- `documentation.html`: long-form project documentation
- `styles/site.css`: shared CSS rules
- `scripts/site.js`: shared JavaScript and jQuery logic

## Stack

- HTML5
- Tailwind CSS via CDN
- External CSS plus page-level `<style>` tags
- JavaScript
- jQuery

## Notes

The project was reworked from a single-page portfolio into a multi-page site so it satisfies the final assignment requirements while staying Tailwind-based.

The full written explanation of the code and user interface lives in `documentation.html`.
